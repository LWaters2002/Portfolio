HOW WE RUST

Collobarative Project

Description

How We Rust is a thematic puzzle game including logic puzzles, parkour and a dystopian sci-fi asthetic.

Software

Unreal Engine, Blender, JIRA

Skills/Topics Explored

JIRA, Project Management, Inverse Kinematics, Animation Targeting and animation driven player movement.

FEATURES

WORKED ON

Parkour Locomotion

Player Vaulting, wall climbing and Ledge grab, all wrapped in a custom character controller with additional features for sliding and climbing ladders. Some of these features use root motion to get accurate and great looking results.

Gravity Boots

Anti-Gravity Boots that stick the character to any nearby magnetic surface, and attract and rotate the player towards it.

Push Pull

Actors in world can be pushed and pulled in different arrangements to help solve puzzles furthermore Target IK's are used to attach the actor's hands to push/pullables anchors.

Interaction and Power System

A system for interacting with the world, such as triggering levers and picking up items. In addition the power system, can share and distribute 'power' to acivate different outputs, this system can use simple logic gates to create designer friendly logic for level and puzzle design.
