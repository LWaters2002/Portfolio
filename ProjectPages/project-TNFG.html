TNFG

Collaborative Project

Description

Totally normal farming game is a student collaborative project with a team of 16+, where I took the role of tech lead. Where the game itself is a farming game, with an arcade-shooter twist.

Software

Unreal Engine, Blender, 3DS Max

Skills/Topics Explored

Blueprint, Unreal Animation System, UMG, JIRA, Task Assignment and Project Management

FEATURES

WORKED ON

Gun System

A designer friendly weapon system that allows for any attribute to be customised easily, the system is built around polymorphism and inheritance to re-use code optimally, for example weapon variants can easily be made such as a charged gun or burst rifle.

Team Lead

During this project I was tech lead, which meant communicating between different disciplines to request and deliver different ideas, mechanics and features. Furthermore this led me to organise and plan tasks around individuals at different skill levels, and utilise a good team workflow through JIRA.

This position allowed me to teach juniors and seniors more about unreal, proper workflow as well as proper conventions when it comes to working in groups.

Scythe

A fully parameterized scythe that acts as a weapon and harvesting tool, that has a special mechanic where bonus damage and bonus harvesting is done during a perfectly timed window.

Build System

An alternative controller for the player that allows them to build and upgrade structures through a world-space context driven menu.

Enemy Feedback

A set of game feel options regarding enemies, such as a custom hit flash material, knockback, timeline driven death animations and sound effects.
