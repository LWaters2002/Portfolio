HYE

SOLO PROJECT

Description

Hye is a roleplaying third person-shooter that allows the player to harness the elements, and make their way through the world, whilst exploring their own expanding gameplay dynamics.

Software

Unity, Blender, Photoshop, DaVinci Resolve

Skills/Topics Explored

Programming, 3D modelling, Texturing, Technical Art, Animation

FEATURES

WORKED ON

Elemental Synergy

A system that allows for numerous effects to be combined and utilised for different gameplay outcomes, allowing a player to experiment with gameplay dynamics and adds an intuitive layer of complexity to the combat.
